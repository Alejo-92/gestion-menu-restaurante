# Gestión de Menú de Restaurante con Promociones

Este proyecto consiste en un programa desarrollado en **Python** que gestiona los platos de un restaurante mediante una matriz y aplica descuentos automatizados de forma inteligente.

## 🚀 Funcionalidades
* **Estructura de Matriz:** Almacena información de platos (Nombre, Categoría y Precio Base).
* **Módulo de Promoción:** Una función evalúa si el producto aplica para un 15% de descuento (basado en su categoría y un umbral de precio).
* **Reporte Automático:** Muestra en pantalla el comparativo de precios base vs. precios finales.

## 🛠️ Tecnologías Utilizadas
* Python 3
* Lógica condicional y estructuras de datos (Listas anidadas/Matrices).
